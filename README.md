## 概要

- 声を出して元気を集めろ！そして弾けろ！

## 操作方法

- 大声を出して元気玉を膨らませる
- 元気満タン状態で**空気の漏れる音(スー)**を出すと元気玉が破裂する（コツを掴むまで難しい）
- これを応用して、**「うぅ〜、すぇいや！」**と声を出して遊ぶと楽しい
- 画面の上端をクリックすると元気メーターをオンオフできる
- それ以外の画面をクリックでリセット
